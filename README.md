## 📈 Sales & Customer Dashboard (2023 Analysis)

![Sales Dashboard](/images/sales_dashboard.png)

[🌐 **View Interactive Dashboard on Tableau Public**](https://public.tableau.com/shared/T3MMYJ232?:display_count=n&:origin=viz_share_link)

This project features a dual-dashboard analysis of 2023 sales performance and customer behavior. The **Sales Dashboard** provides a comprehensive view of revenue, profit, and product-level trends, while the **Customer Dashboard** delivers insights into customer acquisition, order patterns, and top customers by profitability. Together, they offer a complete picture for stakeholders to make data-driven decisions.

**Key Tableau Skills Utilized:**
* 🎨 Dashboard Layout & Design (Multi-Dashboard Navigation)
* ⚙️ Data Preparation & Connection (Multi-Region CSV Datasets)
* 🧮 Calculated Fields & Table Calculations
* 📊 Combo Charts (Bar + Line for YoY Comparison)
* 📈 Sparklines & KPI Trend Indicators
* 📉 Profit & Loss Visual Encoding (Color-Coded)
* 📋 Ranked Tables with Dynamic Sorting
* 📊 Distribution Charts (Histogram / Bar)
* 🔢 KPI Cards with Year-over-Year (YoY) Comparison
* 🎚️ Interactive Filters & Parameters
* 🖱️ Dashboard Navigation Buttons & Custom Icons
* 🎨 Custom Color Palettes & Conditional Formatting
